# 第六单元 — CSS 选择器练习 (3/18)
CSS 练习：编写 class 和 ID 规则并在 HTML 中应用它们。

---

> 💡 先从 `styles.css` 开始 — 然后再打开 `index.html`

---

## 第一部分 — 完全引导

我们给你规则名称和里面的内容。
你的任务是正确地写出语法。

---

**在 `styles.css` 中：**

**任务 1** — 编写一个名为 `pink-bg` 的 class 规则
- 将背景颜色更改为粉色

**任务 2** — 编写一个名为 `blue-text` 的 class 规则
- 将文字颜色更改为蓝色

**任务 3** — 编写一个名为 `big-border` 的 class 规则
- 使用以下值添加边框：`3px solid black`

**任务 4** — 编写一个名为 `main-title` 的 ID 规则
- 将背景颜色更改为深蓝色
- 将文字颜色更改为白色

**任务 5** — 编写一个名为 `footer-text` 的 ID 规则
- 将文字颜色更改为灰色
- 将背景颜色更改为黑色

---

**现在打开 `index.html` 并应用你的规则：**

- 将 `class="pink-bg"` 添加到第一段和最后两段随机文字段落
- 将 `class="blue-text"` 添加到第二段
- 将 `class="big-border"` 添加到第三段和所有五张图片
- 将 `id="main-title"` 添加到 `<h1>`
- 将 `id="footer-text"` 添加到 footer 里的 `<p>`

---

## 第二部分 — 半引导

我们给你规则名称，但不告诉你里面的内容。
你来决定属性和值。
每条规则至少需要有 `background-color` 和 `color`。

---

**在 `styles.css` 中：**

**任务 6** — 编写一个名为 `highlight` 的 class 规则

**任务 7** — 编写一个名为 `list-style` 的 class 规则

**任务 8** — 编写一个名为 `page-heading` 的 ID 规则

---

**现在打开 `index.html` 并应用你的规则：**

- 将 `class="highlight"` 添加到 `<ul>` 列表
- 将 `class="list-style"` 添加到 `<ol>` 列表
- 将 `id="page-heading"` 添加到第二个 `<h2>`

---

## 第三部分 — 开放式

你来决定一切 — 名称和样式。
在 `styles.css` 中编写 **2 条 class 规则** 和 **1 条 ID 规则**。
然后在 `index.html` 中任意应用它们。

祝你好运！🎯
